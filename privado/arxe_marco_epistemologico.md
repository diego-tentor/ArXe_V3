## Sobre Verdad, Prueba y Falsabilidad

### Rechazo del Marco Binario Clásico

**Rechazamos:**
```
Teoría es VERDADERA ⊕ FALSA
Demostración: Derivación que establece verdad absoluta
Falsabilidad: Condiciones experimentales que refutan teoría
```

**Por qué lo rechazamos:**
1. Presupone ontología platónica (existe UNA teoría verdadera esperando ser descubierta)
2. Presupone lógica binaria como única válida (ArXe propone n-ariedades)
3. Colapsa en dogmatismo (teorías defendidas con hipótesis auxiliares ad-hoc)

### Adoptamos Marco Dialógico-Interpretativo

**Proponemos:**
```
Mapeo estructural: Teoría ↔ Fenómeno
Calidad: Coherencia interna + Fertilidad interpretativa + Elegancia
Falsabilidad popperiana: Humildad epistémica, apertura crítica
```

**Criterios de evaluación:**

| Criterio | ¿Qué preguntamos? |
|----------|-------------------|
| **Coherencia interna** | ¿El sistema tiene contradicciones lógicas? |
| **Sistematicidad** | ¿Se aplica consistentemente a múltiples casos? |
| **Parsimonia** | ¿Usa mínimos conceptos primitivos? |
| **Correspondencia** | ¿Mapea razonablemente a observaciones? |
| **Fertilidad** | ¿Genera preguntas nuevas e interesantes? |
| **Iluminación** | ¿Conecta fenómenos antes dispares? |

### Falsabilidad como Actitud, no Experimento

**Somos falsables porque:**
1. ✅ No defendemos ArXe dogmáticamente
2. ✅ Admitimos que podemos estar equivocados
3. ✅ Estamos abiertos a reinterpretación si n(-3) ≠ 7 es mejor mapeo
4. ✅ No creamos hipótesis auxiliares ad-hoc para "salvar" predicciones fallidas
5. ✅ Nos comprometemos al diálogo racional crítico

**NO somos falsables en sentido positivista** porque:
- No hay "experimento único que refuta ArXe"
- Esto presupondría juego de verdad binaria que rechazamos
- Un error predictivo nos invita a reinterpretar, no a "abandonar la teoría"

## Sobre Predicciones y Error

### Error No Es "Falla", Es "Información"

**En marco binario:**
```
Predicción ≠ Medida → TEORÍA REFUTADA
```

**En ArXe:**
```
Predicción ≠ Medida → Analizar estructura del error
                    → ¿Error tiene estructura prima?
                    → ¿Revela operadores faltantes?
                    → ¿Ilumina el fenómeno?
```

**Ejemplo real:**
```
ArXe predijo: m_t = 11,700 GeV
Experimento: m_t = 173 GeV
Ratio: 11,700/173 = 67.6 ≈ 68 = 2²×17

¡El error tenía estructura prima!
Revelaba factor faltante: "doble simetría espectral"

Fórmula corregida: m_t = [predicción]/(2²×17)
Nuevo error: 0.6% ✓
```

## Estándar de Rigor

### Aunque rechazamos marco binario, mantenemos rigor:

**1. Coherencia lógica**
- Axiomas claros (¬() ≜ Tf)
- Derivaciones explícitas (ExEnt → n(k))

**2. Sistematicidad**
- Reglas aplicadas consistentemente
- Criterios de mapeo explícitos

- Errores cuantificados
- Comparación con datos precisos
- Honestidad sobre discrepancias

**4. Parsimonia**
- Conceptos primitivos mínimos
- No multiplicar entidades sin necesidad
- Simplicidad donde sea posible

**5. Fertilidad**
- Genera preguntas nuevas
- Conecta fenómenos dispares
- Sugiere investigación futura
