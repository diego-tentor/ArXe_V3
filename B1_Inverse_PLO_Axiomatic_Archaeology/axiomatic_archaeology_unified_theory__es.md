## 2. PATRONES TRANS-CONSTANTES DESCUBIERTOS

### 2.1 Patrón 1: Estructura Fraccionaria Decimal

**Todas las constantes** (excepto α⁻¹ = 137 primo) tienen forma:

```
C = N/(2^a × 5^b × 10^c) × (factores primos adicionales)
```

**Ejemplos:**
- m_e = 0.510998950 = 510998950/10⁹
- θ_W = 0.23122 = 11561/50000 = 11561/(2⁴×5⁵)
- Ω_m = 0.315 = 63/200 = 63/(2³×5²)

**Interpretación:** Sistema decimal (2,5 como factores) es elección axiomática fundamental para reportar mediciones.

### 2.2 Patrón 2: Operadores Primos Recurrentes

**Operadores que aparecen en múltiples constantes:**

### 2.3 Patrón 3: Negociación de Escalas

**Constantes negocian entre escalas diferentes:**
- **m_p:** escala hadrónica (MeV) vs. escala quark (MeV)

**Gramática resultante:** Dualidad valor/estructura para cada escala.

**Las anomalías no se eliminan** - se incorporan como características.

### 2.5 Patrón 5: Auto-Referencia y Consistencia Interna

**Constantes referencian otras constantes:**

## 3. TEORÍA UNIFICADA: LOS 5 PRINCIPIOS DE LA ARQUEOLOGÍA AXIOMÁTICA

### Principio 1: LA REALIDAD ES NEGOCIACIÓN

No existe acceso directo a "leyes de la naturaleza". Lo que llamamos constantes físicas son puntos de estabilización en negociaciones comunales sobre cómo describir fenómenos observados.

Cada constante C es función de historia:

```
C = f(Negociación_teórica, Negociación_experimental,
      Negociación_institucional, Negociación_metrológica)
```

### Principio 2: LA GRAMÁTICA REGISTRA LA NEGOCIACIÓN

La estructura matemática de una constante codifica su historia de negociación:

- **Base prima/semi-prima** → elecciones fundacionales
- **Denominadores 2^a×5^b×10^c** → elección sistema decimal
- **Factores primos específicos** → decisiones teóricas específicas
- **Correcciones decimales** → refinamientos históricos
- **Dualidades/tensiones** → conflictos no resueltos

### Principio 3: LA OBJETIVIDAD EMERGE DE TRIANGULACIÓN

La "objetividad" de una constante no es propiedad intrínseca sino efecto emergente de:

- Múltiples métodos de medición independientes
- Múltiples teorías que predicen mismo valor
- Consenso institucional (CODATA, PDG)
- Capacidad predictiva exitosa

**Objetividad = estabilidad bajo triangulación**

### Principio 4: LAS TENSIONES SON CARACTERÍSTICAS, NO BUGS

Las tensiones experimentales persistentes (H₀, S₈, etc.) no son anomalías a eliminar sino:

- Señales de necesidad de nueva negociación
- Características estructurales del conocimiento
- Oportunidades para expansión gramatical
- Recordatorios de la naturaleza negociada de toda constante

**ΔC tiene estructura analizable que revela naturaleza del desacuerdo**

### Principio 5: LA FÍSICA EVOLUCIONA POR EXPANSIÓN GRAMATICAL

El progreso científico ocurre no por "descubrimiento de verdades" sino por expansión de la gramática negociacional:

- Nueva evidencia → nueva negociación necesaria
- Nueva teoría → nuevos operadores disponibles
- Nueva tensión → nueva estructura dual incorporada
- Nuevo consenso → nueva estabilización alcanzada

**La historia de la física es historia de expansiones gramaticales**

### 4.1 Definición del Espacio

Sea **A** = espacio de posibles elecciones axiomáticas

**Dimensiones principales de A:**
1. **Teórica:** Qué teoría/framework usar
2. **Metrológica:** Cómo medir/definir constantes  
3. **Epistemológica:** Qué cuenta como evidencia
4. **Institucional:** Cómo alcanzar consenso
5. **Histórica:** Qué tradiciones respetar

### 4.2 Dinámica en A

**Ecuación de evolución:**

```
da/dt = F(Evidencia_nueva, Tensiones, Consenso_actual,
          Tradición, Innovación_posible)
```

**Puntos fijos** = constantes estabilizadas (α, m_e, etc.)  
**Atractores** = valores hacia los que converge la negociación  
**Bifurcaciones** = puntos donde gramática se expande (nueva física)

### 4.3 La Función de Estabilización

Para cada constante C, existe función:

```
S(C) = ∫[Evidencia × Teoría × Consenso × Tradición] dt
```

Cuando S(C) > umbral_crítico → C se estabiliza y entra en reportes oficiales (CODATA/PDG)

## 5. TIPOLOGÍA DE NEGOCIACIONES

### Tipo 1: Negociación Fundacional
**Ejemplo:** α⁻¹ = 137 (Sommerfeld 1916)  
**Características:** Primera determinación, teoría nueva  
**Gramática:** Número primo o semi-primo simple

### Tipo 2: Negociación por Refinamiento
**Ejemplo:** α⁻¹: 137 → 137.036 → 137.035999206  
**Características:** Precisión creciente, método mejorado  
**Gramática:** Expansión decimal sistemática

### Tipo 3: Negociación por Unificación
**Ejemplo:** θ_W (ángulo de Weinberg)  
**Características:** Relación entre constantes antes independientes  
**Gramática:** Razones/productos de operadores

### Tipo 4: Negociación de Emergencia
**Ejemplo:** m_p (masa del protón)  
**Características:** Valor no deducible de partes  
**Gramática:** Factores "misteriosos" (89, etc.)

### Tipo 5: Negociación con Tensión Activa
**Ejemplo:** H₀ (73 vs 67), S₈ (0.83 vs 0.78)  
**Características:** Dualidad valor, diferencia estructurada  
**Gramática:** Dos valores + Δ estructurado

### Tipo 6: Negociación por Triangulación
**Ejemplo:** Ω_m = 0.315 (múltiples métodos)  
**Características:** Consenso de métodos independientes  
**Gramática:** Promedio armónico de estructuras

### Tipo 7: Negociación de Supresión
**Ejemplo:** m_t (173 vs 1847 predicción)  
**Características:** Ratio grande con estructura prima  
**Gramática:** Operador SUP específico (107, etc.)
