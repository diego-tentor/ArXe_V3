# RESUMEN DE CONSTANTE DE HUBBLE (H₀)

## DATOS BASE
- **Constante**: Constante de Hubble H₀
- **Valores en tensión**:
  - **Local (SH0ES)**: 73.04 ± 1.04 km/s/Mpc
  - **Cosmológico (Planck)**: 67.36 ± 0.54 km/s/Mpc
- **Diferencia**: 5.68 km/s/Mpc (~5.5σ tensión)
- **Unidades**: km/s/Mpc

## TENSIÓN PRINCIPAL
**Dos valores en conflicto**:
- **Métodos locales** (escalera de distancia): Cefeidas, supernovas Ia → ~73.04
- **Métodos cosmológicos** (CMB + BAO): Planck, oscilaciones bariones → ~67.36
- **Diferencia**: 5.68 km/s/Mpc (7.8% diferencia relativa)

## HISTORIA DE DECISIONES AXIOMÁTICAS (11 decisiones principales)
1. **1929**: "El universo se expande, tasa constante H" → Concepto H (sin valor preciso)
2. **1952**: "Cefeidas como candelas estándar primarias" → Factor de calibración periódica-luminosidad
3. **1958**: "Sandage: H₀ ~ 75 (valor alto)" → 75 = 3×5² = CYC×MEM²
4. **1975**: "Sandage revisa a 55 (valor bajo)" → 55 = 5×11 = MEM×REG
5. **1990**: "HST permite mediciones extragalácticas precisas" → Factor precisión ~10× mejor
6. **1999**: "Key Project establece ~72 como referencia" → 72 = 2³×3² = EXP³×CYC²
7. **2006**: "SH0ES: ~73 con supernovas + cefeidas" → 73 = OSC (primo 21º) emerge
8. **2013**: "Planck: ~67.3 del CMB establece cosmología precisa" → 67.36 = SCAT + CYC²/MEM²
9. **2018**: "SH0ES: ~73.04 se consolida como valor local" → 73.04 = OSC + 1/MEM²
10. **2020**: "La tensión ~5.7 se acepta como problema real" → Δ = 5.68 = MEM + SPEC/MEM²
11. **2023**: "Dos valores coexisten sin resolución" → Dualidad como característica

## ESTRUCTURAS GRAMATICALES DUALES

### H₀_local (SH0ES)
73.04 = 73 + 0.04 = OSC + 1/MEM²
- **73 = OSC**: Primo 21º (oscilaciones)
- **1/MEM²**: 1/25 = 1/5²

### H₀_cmb (Planck)
67.36 = 67 + 0.36 = SCAT + CYC²/MEM²
- **67 = SCAT**: Primo 19º (scattering)
- **CYC²/MEM²**: 9/25 = 3²/5²

### Diferencia (Tensión)
ΔH₀ = 5.68 = 5 + 0.68 = MEM + SPEC/MEM²
- **5 = MEM**: Memoria/calibración histórica
- **SPEC/MEM²**: 17/25
